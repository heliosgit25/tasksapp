This is my first Python project. 

This is a simple CLI Tasks App.

To use this app, save the tasksapp.py file in a location of your choice. Save the .command file as well in the same location.
Inside the .command file replace the filepath with the filepath where you have saved the above two files.

Go to system settings -> General -> Login items & extensions -> Add the .command file under "Open at login" section.

Now, everytime you start up your Mac, the app will run in your terminal.

Note- Apple doesn’t officially support unlock triggers. Hence, it is difficult to trigger the app to launch everytime you unlock your Mac. But feel free to create your own LaunchAgent and use the app every time you unlock your Mac.
